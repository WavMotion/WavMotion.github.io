物品id为10进制，CE中需输入2个16进制的ID

计算方法：

ID 1=“物品id”/ 256 的余数 并将结果从10进制换算成16进制；

ID 2=“物品id”/256 的商 并将结果从10进制换算成16进制；

以耐毒珠为例：

物品ID=727

727/256=2 余 215

215换算成16进制是D7

2换算成16进制是2

所以ID 1=D7 , ID 2 = 2
